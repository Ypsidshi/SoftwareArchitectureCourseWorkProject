# Агенты и ИИ в репозитории диплома

Репозиторий: **ИС клиентского бронирования** (Go, PostgreSQL, NATS, микросервисы). Основная реализация — `src/go-microservices`.

## Роли настроек

| Настройка | Назначение |
|-----------|------------|
| `.cursorrules` | Go, микросервисы, Clean Architecture — **не перезаписывать** |
| `CLAUDE.md` | ПЗ, стиль, PlantUML, pandoc — полный справочник по тексту |
| `.cursor/rules/*.mdc` | Контекстные правила Cursor (ПЗ, диплом, код) |
| `.cursor/skills/*/SKILL.md` | Пошаговые сценарии (см. `.cursor/README.md`) |
| `.cursor/scripts/` | Вспомогательные проверки (нумерация рисунков) |

## Ключевые пути

### Диплом (актуально)

- Понимание проекта: `docs/project-understanding-guide.md`

## Когда какой скилл

| Запрос пользователя | Скилл |
|---------------------|--------|
| Чего не хватает, готовность к защите, сверка кода и audit | `diploma-gap-audit` |
| Объясни, как устроено, схема, разбор потока | `diploma-explain` |
| Новый текст ПЗ диплома (только по явному запросу и новому файлу) | `diploma-pz-workflow` |
| fixes.md / CourseworkCurrent — **не применять** без явного запроса | — |
| Вопросы комиссии, демо, ограничения | `diploma-defense-prep` |

Перед ответом по диплому **читать** релевантные файлы из таблицы; не выдумывать API и разделы ПЗ.

## Типовой цикл работы

1. **Первый заход** — `diploma-gap-audit` → отчёт с приоритетами.
2. **Понять систему** — `diploma-explain` + `docs/project-understanding-guide.md` (+ `project-self-learning-roadmap.md` для самообучения).
3. **Доработать проект под диплом** — код, `project-readiness-audit.md`; текст ПЗ диплома — отдельно, когда будет готов документ.
4. **Перед защитой** — `diploma-defense-prep` + повторный краткий аудит.

Комбинированный запрос («разберись и скажи что доделать») = аудит, затем объяснение только по найденным пробелам.

## Устаревшее в репозитории

- `src/services`, `src/gateway` — черновик .NET, **не** основная система для защиты.
- При расхождении сданной курсовой (`CourseworkCurrent.md`) и кода — зафиксировать в ответе; **не править** сданный текст, дорабатывать код или будущий текст диплома по запросу.

## Язык

Ответы пользователю — **на русском**. Текст ПЗ — официально-деловой стиль (см. `CLAUDE.md` и правило `coursework-pz`).
