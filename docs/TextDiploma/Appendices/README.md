# Приложения к пояснительной записке

| Файл | Назначение |
|:-----|:-----------|
| `Appendices.md` | Приложения А, В–Л (глоссарий-заглушка, код, API-скрины, Ганта) |
| `Appendices.docx` | Сборка для Word |
| `Appendix_B_Specification.md` | Приложение Б — ключевые таблицы БД (Б.1–Б.5) |
| `Appendix_B_Specification.docx` | Сборка приложения Б (см. команду ниже) |

В конце готовой ПЗ приложения идут **после списка источников**.

Образцы оформления: `../Examples/Appendices/`.

**Сборка DOCX** — без `--reference-doc` (шаблон ломает таблицы в Word):

```powershell
Set-Location docs/TextDiploma/Appendices
pandoc Appendix_B_Specification.md -o Appendix_B_Specification.docx --from markdown --to docx
pandoc Appendices.md -o Appendices.docx --from markdown --to docx
```

Глоссарий (приложение А) может правиться напрямую в Word; `Appendices.md` не пересобирать в docx без необходимости, чтобы не затереть ручные правки.
