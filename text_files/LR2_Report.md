## Объект тестирования
Фрагмент бизнес-логики микросервиса (Go) для оценки бронирования:

```go
// CheckBooking оценивает заявку на бронирование
// nights  – длительность проживания
// guests  – число гостей
// highSeason – флаг высокого сезона
func CheckBooking(nights int, guests int, highSeason bool) string {
    status := "OK"
    if (nights > 14) && (guests > 2) { // b1 & b2
        status = "Need manager approval"
    }
    if (guests == 1) || highSeason {   // d1 v d2
        status = status + " / Prepay required"
    }
    return status
}
```

Условные метки: b1 = nights > 14; b2 = guests > 2; d1 = guests == 1; d2 = highSeason == true.
