### 2.3 Расчёт показателей Холстеда

- `n1 = 19` — число различных операторов
- `N1 = 130` — общее число операторов
- `n2 = 46` — число различных операндов
- `N2 = 104` — общее число операндов
- `n = n1 + n2 = 65`
- `N = N1 + N2 = 234`
- `V = N * log2(n) = 234 * log2(65) = 1409.23`
- `D = (n1 / 2) * (N2 / n2) = (19 / 2) * (104 / 46) = 21.48`
- `E = D * V = 21.48 * 1409.23 = 30267.90`

Вывод: объём `V > 1000`, что указывает на достаточно высокую информационную насыщенность метода.
